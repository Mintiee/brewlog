# brewlog

A mobile-first web app for logging pour-over coffee and managing your bean shelf —
track every brew, see which beans are at their peak, and learn what your palate likes.

**👉 Live app: [coffeebrewlog.vercel.app](https://coffeebrewlog.vercel.app)**

Built with Next.js + Supabase. Optional bring-your-own AI key adds bag-photo / link
import and palate insights; everything else works without it.

Self-hosting? See **[DEPLOY.md](./DEPLOY.md)**.
